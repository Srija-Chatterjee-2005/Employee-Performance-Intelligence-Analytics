# AI-Powered Employee Performance Intelligence System

A recruiter-ready machine learning and analytics project that predicts employee performance using synthetic HR data and displays business insights in an interactive Streamlit dashboard.

## Features
- Synthetic HR dataset generator
- Employee performance prediction using machine learning
- Multi-model comparison
- Interactive filters and tabs
- HR recommendations for training, promotion, and retention
- Visual analytics dashboard

## Run
```bash
pip install -r requirements.txt
streamlit run main.py
```
